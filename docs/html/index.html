Overview¶

solar.charge

This module collects data from E3DC System via Modbus-Interface.

solar.car

class Car contains all Tesla car data and methods to access car Methods for self.func are imported from class Vehicle in module teslaapiclient.teslavehicle

solar.send_status

Function to send emails about the status or errors to user.

solar.chargemodbus

Class ChargeModbus with methods to collect and prepare data from E3DC modbus interface

solar.definitions.pvdataclasses

Module containing different dataclasses with some default values.

solar.definitions.send_status_access

Access data for Email notification Please fill in correct data and rename file to ‘se’

solar.definitions.e3dc_register

Internal mapping data for E3DC modbus based on:

solar.definitions.access_data

Mandatory access data for E3DC and Tesla Car

solar.teslaapi.teslaapiclient

Class TeslaApiClient with methods for authentification and get and post methods for access to Tesla API

solar.teslaapi.teslavehicle

class used by class Car with method to manipulate car
